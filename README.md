# Wiki Springfield

## Descripción
Wiki Springfield es un sitio web estático que presenta información sobre personajes de Los Simpson. El proyecto incluye una página de inicio con tarjetas de personajes y páginas internas que muestran detalles específicos de Homero y de la familia Simpson, manteniendo un estilo coherente y responsive.

## Tecnologías utilizadas
- 🟦 **HTML5** — Estructura y contenido de las páginas.
- 🎨 **CSS3** — Estilos personalizados para el diseño visual.
- ⚡ **Bootstrap 5** — Componentes y diseño responsive.
- 🌐 **Arquitectura estática** — Estructura simple basada en archivos locales.

## Características principales
- Navegación consistente entre páginas.
- Diseño responsivo con Bootstrap.
- Páginas separadas para personajes y familia.
- Formulario de registro simple en la sección de familia.
- Diseño basado en cards y una presentación visual clara.

## Estructura del repositorio
- index.html — Página principal del proyecto.
- homero.html — Página del personaje Homero Simpson.
- familiasimpson.html — Página de la familia Simpson.
- styles.css — Estilos globales y personalizados.
- img — Imágenes utilizadas en el sitio.

## Uso
1. Clona o descarga el repositorio.
2. Abre index.html en un navegador web.
3. Navega entre la página principal y las páginas internas usando el menú.
